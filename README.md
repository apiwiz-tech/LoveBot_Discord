# LoveBot

## Setup Instructions

1. Fork this repository.
2. On Render.com, create a new Web Service:
   - Connect GitHub repo.
   - Set Environment Variables:
     - `DISCORD_TOKEN`: your bot token
     - `OPENAI_KEY`: your OpenAI API key
     - `GUILD_ID`: your private server ID
3. Deploy and start.
4. Invite the bot to your server with `applications.commands` scope.

## Usage
- Use `@LoveBot summary` to get weekly summaries.
- Use `@LoveBot highlights` to see affectionate messages.
- `@LoveBot why-fight` to analyze conflicts.

Enjoy your AI relationship companion! ❤️
